Using Fonts

If you can, its always great to try and use a non-serrfied font in your design. In places you need to use the logo without the mark, always use the official font "Righteous". (download). Never combine "COFFEE OUTLET" with any other words and always make sure there is enough padding around it so it stands out on its own as a logo.

On print, the website, and other digital mediums we also use fonts "Lato" and "Montserrat" for larger blocks of text. You can get Lato and Montserrat here; and both are free and open source and can be found on Google Fonts to link to for use in web. For some signage and headings that need all caps, you can use Montserrat Subrayada (Download Here).
